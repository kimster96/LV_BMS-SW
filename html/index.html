Requirements

The LV BMS has a set of requirements imposed by both the FSG rules and internal requirements. The summary of requirements is lested as follows

FSG Rules requirements:

  • Monitor all cell voltages
  • Monitor >30% of cell temperatures
  • Monitor the output current
  • Transmit the above measurements through CAN
  • Interrupt the output if the voltages exceed the limits for more than 500 ms
  • Interrupt the output if the temperatures exceed the limits for more than 1 s

Internal requirements:

  • Configurable thresholds via CAN messages
  • Store thresholds on non volatile memory and retreive the on power-up
